# True Stay Insights

True Stay Insights is a premium guest customer feedback software designed for the hospitality industry.

## About the Platform

TSI (True Stay Insights) utilizes advanced AI to capture genuine guest feedback through natural voice conversations. By automatically calling guests after checkout, we replace traditional, low-response email surveys with actionable, real-time insights.

**Key Features:**
-   **AI Voice Calls:** Natural, conversational feedback collection.
-   **Sentiment Analysis:** Understand the emotional tone of your guests.
-   **Real-time Escalations:** Detect and alert staff to negative experiences immediately.
-   **Dashboard Reporting:** Comprehensive analytics on guest satisfaction.
