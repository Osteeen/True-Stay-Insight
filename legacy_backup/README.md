# TrueStay Insight

**Premium Guest Feedback Automation for Hospitality**

TrueStay Insight automatically calls your guests after checkout to capture natural feedback, detect issues early, and prevent negative public reviews. Built with React, TypeScript, and Framer Motion for a premium user experience.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
